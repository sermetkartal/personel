# Personel — Eğitim Video Scriptleri

Bu dizin, Personel Platform için hazırlanacak 3 tanıtım/eğitim videosunun
script'lerini içerir. Videolar henüz çekilmemiştir — bu dosyalar placeholder
niteliğindedir ve production ekibine devredilmeye hazırdır.

## Videolar

1. **01-personel-tanitim.md** — 3 dakika, Personel nedir + neden kullanılır (satış videosu)
2. **02-admin-quickstart.md** — 10 dakika, admin Console'a ilk girişten ilk rapora kadar
3. **03-kvkk-dpo-walkthrough.md** — 15 dakika, DPO için KVKK workflow tur

## Format

Her script:

- **Sahne ayrımı** (30-60 saniyelik bloklar)
- **Ekran gösterimi** talimatları (hangi UI elementi, hangi hareket)
- **Ses tonu** (sakin, kurumsal, Türkçe ana)
- **Arka plan müziği** önerisi (royalty-free)
- **Altyazı** (TR + EN)

## Üretim Notları

- Çekim ortamı: Pilot vm3 (dev-seed-showcase.sh ile hazırlanmış)
- Yazılım: Camtasia / OBS + Adobe Premiere (veya DaVinci Resolve)
- Süre toplamı: 28 dakika
- Tahmini maliyet: ₺25-40K (stüdyo + düzenleme + ses)
- Teslim süresi: 4-6 hafta (script onayından sonra)

## Yayın

- Internal: `docs.personel.local/videos/`
- External: YouTube (unlisted, kanaldan paylaşım)
- Müşteri portalı: gömülü (iframe)
